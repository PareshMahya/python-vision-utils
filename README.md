# Play With Python

This repository is a personal Python playground for learning, experimenting, and building small utilities and prototypes.  
It includes short scripts, helper modules, and mini projects covering topics such as data processing, automation, and computer vision.

## Purpose
- Practice Python programming
- Experiment with libraries and tools
- Build reusable utility scripts
- Learn through small, focused examples

## Contents
- face_emotion_utils: face detection and emotion recognition utilities
- notebooks: Jupyter notebooks for experiments and analysis
- scripts: standalone Python scripts
- utils: reusable helper modules

## Technologies Used
- Python 3
- OpenCV
- DeepFace
- NumPy
- Pandas

## Usage
Create a virtual environment and install dependencies:

pip install -r requirements.txt

Run scripts from the scripts folder or import utilities from the utils folder.
